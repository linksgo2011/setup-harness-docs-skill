关于 {{PROJECT_NAME}} 的开发记录。此目录包含规范、计划、架构决策与设计事实。

## 目录用途

| 目录 | 用途 |
|------|------|
| `standards/` | 开发规范（layers、db、api、security、naming、error、test 等） |
