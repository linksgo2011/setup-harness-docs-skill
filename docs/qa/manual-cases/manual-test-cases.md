## 3. 用户预约流程

### TC-M-07: 完整预约流程

| 项目 | 内容 |
|------|------|
| 前置条件 | 已注册并登录 |
| 步骤 | 1. 点击导航栏"预约咨询"或仪表盘"立即预约"<br>2. Step 1：点击选择一个咨询师（如张医生）<br>3. Step 2：点击选择一个日期<br>4. Step 3：点击选择一个时间 slot（如 09:00）<br>5. Step 4：可选填写备注，点击"确认预约" |
| 预期结果 | - 预约成功<br>- 跳转到"我的预约"页面<br>- 列表中显示新创建的预约，状态为"待确认" |

| 预期结果 | - 预约状态变为"已取消"<br>- 取消按钮消失 |
